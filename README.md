# 日常记账工具

可以在5秒之内记录一笔收支，且无需担心数据丢失或被非法利用。目前是为满足我自己日常快速记账的需求而开发的小工具，也是为了替代某个我使用已久的第三方记账App。

- 基于Next.js开发
- 支持离线使用，可添加到手机桌面
- 支持多端同步
- 记录保存在指定的Github仓库中
- 仅支持单用户
- 方便部署且无成本
